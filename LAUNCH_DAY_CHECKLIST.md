## 🚨 Common Issues & Quick Fixes

### Issue: Low Initial Upvotes (< 50 in first hour)
**Fix**:
- Share more actively on Twitter (tag relevant accounts)
- Alert beta users (don't ask for upvotes, just share the launch)
- Engage in PH comments on other products (visibility)
- Check if hunter posted (if using hunter)

### Issue: Negative Comment About ToS Violation
**Fix**:
- Respond immediately with Template (see PRODUCT_HUNT_LAUNCH_PLAN.md)
- Emphasize: Only reads public data, no booking automation
- Offer to discuss privately if they have concerns
- Stay calm and professional

### Issue: Extension Crashes During Launch
**Fix**:
- Post transparent update in PH comments
- Provide timeline for fix
- Offer workaround if possible
- Fix ASAP, update when resolved
- Turn crisis into trust-building moment

### Issue: Chrome Store Link Not Working
**Fix**:
- Check if extension ID is correct in landing page
- Verify Chrome Web Store listing is live (not in review)
- Update all links immediately
- Post correction in PH comments
- Apologize and provide working link

### Issue: Competitor Launches Same Day
**Fix**:
- Stay positive, don't bash competitor
- Focus on unique differentiation (sound alerts, slot history, free tier)
- Engage with their audience authentically
- Offer to collaborate if genuine overlap

## 📱 Response Templates (Quick Copy-Paste)

### "Is this legal?"
```
Great question! NEXUS Alert only monitors publicly available appointment data on ttp.cbp.dhs.gov — the same data anyone can see by visiting the website. We don't automate booking, create accounts, or interact with the system beyond reading availability. It's equivalent to manually refreshing the page, just automated for convenience.
```

### "How is this different from X?"
```
Key differences:
1. Built specifically for GOES system (NEXUS/Global Entry/SENTRI)
2. Smart filtering by date, time, and location
3. Sound alerts so you don't have to watch your screen
4. Slot history to identify patterns
5. One-click booking link
6. Free tier (most competitors are paid-only)

What specific features are you looking for?
```

### "Can you add [feature]?"
```
Love this idea! We're tracking feature requests in our GitHub repo. Can you open an issue at github.com/caffeineGMT/nexus-alert/issues so we can discuss details and prioritize it? Really appreciate the feedback!
```

### "Does it work on Firefox/Safari?"
```
Currently Chrome/Edge/Brave only (Chromium-based browsers). We're considering Firefox support based on demand — if you'd use it, let us know in our GitHub issues!
```

### General Thank You
```
Thank you so much for the support! 🙏 Let me know if you have any questions about the extension.
```
