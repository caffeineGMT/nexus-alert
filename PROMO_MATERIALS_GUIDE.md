## 🎥 Promo Video (30-60 seconds)

### Video Specs
- **Resolution**: 1920x1080 (1080p) or 1280x720 (720p)
- **Format**: MP4 (H.264 codec)
- **Max Size**: 20MB
- **Length**: 30-60 seconds (optimal: 45 seconds)
- **Frame Rate**: 30fps or 60fps
- **Audio**: Background music (low volume) + sound effects

### Video Script (45-second version)

```
[00:00-00:05] HOOK - Problem
Visual: Split screen - GOES website showing "Next available: August 2025" vs Calendar
Text Overlay: "Wait 6+ months for a NEXUS appointment?"
Voiceover: "NEXUS appointments are booked months out..."

[00:06-00:10] Problem Detail
Visual: Person manually refreshing browser, looking frustrated
Text Overlay: "Cancelled slots appear randomly and disappear in minutes"
Voiceover: "...but cancelled slots appear randomly and disappear in minutes."

[00:11-00:17] Solution Introduction
Visual: Chrome Web Store → Install extension → Extension popup appears
Text Overlay: "NEXUS Alert monitors 24/7"
Voiceover: "NEXUS Alert monitors appointment slots automatically, so you don't have to."

[00:18-00:28] Demo - Key Features
Visual: Extension interface showing:
- Multiple locations selected
- Check frequency: Every 2 minutes
- Date filter: March 1 - April 30
Then: Notification fires (with sound effect!)
Desktop notification: "NEXUS Appointment Found! Niagara Falls - March 15 @ 2PM"
Voiceover: "Choose your preferred locations, dates, and times. When a slot opens, you get instant alerts with sound."

[00:29-00:35] Booking Flow
Visual: Click notification → GOES booking page opens → Appointment booked
Text Overlay: "One click to book"
Voiceover: "One click takes you straight to booking. Fast, simple, effective."

[00:36-00:42] Results
Visual: Slot history showing 15+ found appointments
Text Overlay: "Found 15 slots in 7 days"
Voiceover: "Real users are finding slots within days instead of waiting months."

[00:43-00:45] CTA
Visual: Extension icon + Product Hunt badge
Text Overlay: "Install free. Product Hunt Special: First month premium free."
URL: nexusalert.com/ph
Voiceover: "Try NEXUS Alert free today."
```

### Recording Tools

**Screen Recording**:
- **Mac**: QuickTime (free), ScreenFlow ($169), Camtasia ($299)
- **Windows**: OBS Studio (free), Camtasia ($299)
- **Browser-based**: Loom, Screen Studio

**Video Editing**:
- **Beginner**: iMovie (Mac, free), DaVinci Resolve (free)
- **Intermediate**: Adobe Premiere Rush, Final Cut Pro
- **Professional**: Adobe Premiere Pro, Final Cut Pro X

**Stock Assets**:
- **Music**: Epidemic Sound, Artlist, YouTube Audio Library (free)
- **Sound Effects**: Freesound.org, Zapsplat (free)
- **Stock Video**: Pexels, Unsplash (free)

### Video Production Checklist

- [ ] Write detailed script with timings
- [ ] Record screen captures (extension demo, notifications, booking flow)
- [ ] Record voiceover (or use text overlays only)
- [ ] Select background music (royalty-free, low volume)
- [ ] Add sound effects (notification chime, click sounds)
- [ ] Add text overlays for key points
- [ ] Add captions/subtitles (80% watch with sound off)
- [ ] Export at 1080p, H.264, MP4
- [ ] Compress to under 20MB (Handbrake, Adobe Media Encoder)
- [ ] Test on mobile and desktop
- [ ] Upload to Product Hunt

## 🎯 Quick Tips

### Gallery Images
✅ **Do**:
- Use high-resolution screenshots
- Add subtle annotations/arrows
- Show real, working product
- Keep text minimal and large
- Use consistent branding (colors, fonts)

❌ **Don't**:
- Use lorem ipsum or fake data
- Over-annotate or clutter
- Use low-quality screenshots
- Add too much text
- Stray from brand identity

### Promo Video
✅ **Do**:
- Show the product in action
- Keep it short (30-60 seconds)
- Include captions (accessibility + silent viewing)
- Use real sound effects (notification chime)
- End with clear CTA

❌ **Don't**:
- Talk too fast
- Use copyrighted music
- Exceed 20MB file size
- Skip the hook (first 3 seconds critical)
- Forget mobile optimization

### Thumbnail
✅ **Do**:
- Keep it simple
- Use high contrast
- Make it recognizable at small size
- Use brand colors

❌ **Don't**:
- Add detailed text
- Use busy backgrounds
- Deviate from extension icon
- Forget to test at 240x240px
