The Problem

NEXUS Appointments Are Nearly Impossible to Book

Since the pandemic, Canadian NEXUS enrollment centers have been non-operational for in-person interviews. The few U.S. border locations that remain open see appointment slots disappear within seconds of becoming available. Hundreds of thousands of conditionally-approved applicants are stuck in limbo.

The Current Experience

Applicants must manually refresh the TTP Scheduler website dozens of times per day, hoping to catch a cancellation. There is no official notification system. Slots appear and vanish in seconds. Working professionals and families cannot realistically monitor the site 24/7. The result: a broken system where only the most persistent (or automated) users succeed.

How It Works

Three Steps to Never Missing a Slot

NEXUS Alert runs quietly in your browser, polling the official TTP Scheduler API and alerting you the moment a slot opens at your chosen locations.

1

Install the Extension

Add NEXUS Alert from the Chrome Web Store. Select your preferred enrollment center locations and configure notification preferences. Setup takes under 60 seconds.

2

Automated API Polling

The extension's Service Worker polls the public TTP Scheduler API at configurable intervals (default: every 60 seconds). It checks for available appointment slots at your selected locations using the official /schedulerapi/slots endpoint.

3

Instant Notifications

When a slot appears, you receive a Chrome desktop notification with the date, time, and location. Click the notification to jump directly to the TTP booking page and claim your appointment before it disappears.

Key Technical Details

Public API Endpoints

The TTP Scheduler exposes a fully public, unauthenticated REST API. No API key required. These endpoints power the entire NEXUS Alert system.

GET — Available Appointment Slots

// Check for available slots at a specific location GET https://ttp.cbp.dhs.gov/schedulerapi/slots ?orderBy=soonest &limit=1 &locationId=5020 &minimum=1 // Response (200 OK) [{ "locationId": 5020, "startTimestamp": "2026-04-15T08:00", "endTimestamp": "2026-04-15T08:15", "active": true, "duration": 15, "remoteInd": false }] // Empty array [] = no slots available

GET — Enrollment Center Locations

// Fetch all operational NEXUS enrollment centers GET https://ttp.cbp.dhs.gov/schedulerapi/locations/ ?temporary=false &inviteOnly=false &operational=true &serviceName=NEXUS // Returns array of location objects with id, name, address, etc.

GET — Slot Count by Location

// Get count of available slots at a location GET https://ttp.cbp.dhs.gov/schedulerapi/slot-count ?locationId=5020 // Useful for dashboard displays and historical tracking

Existing Solutions & Their Limitations

GitHub Scripts & Bots

Require Python, Docker, or Node.js setup. Not accessible to non-technical users. No Chrome integration. Must run on a separate machine or server. Examples: trusted-traveler-scheduler, global-entry-slot-notifier.

Paid Web Services

AppointmentScanner charges $19.95 for email/SMS alerts. GoGlobalEntry is free but ad-supported and slow. Neither offers desktop push notifications or browser-native integration. Both require account creation.

Legal & Safety

Compliance & Responsible Use

NEXUS Alert is designed to operate within legal and ethical boundaries while providing maximum value to users navigating a broken appointment system.

What NEXUS Alert Does

  • ✓ Reads publicly available, unauthenticated API data
  • ✓ Polls at reasonable intervals (minimum 30 seconds)
  • ✓ Notifies users to manually book via the official TTP website
  • ✓ Stores all user preferences locally in the browser
  • ✓ Open-source core for full transparency

What NEXUS Alert Does NOT Do

  • ✗ Does not auto-book appointments on behalf of users
  • ✗ Does not access or store TTP login credentials
  • ✗ Does not scrape or bypass any authentication
  • ✗ Does not hoard or resell appointment slots
  • ✗ Does not violate the Computer Fraud and Abuse Act (CFAA)

Legal Basis

The TTP Scheduler API at ttp.cbp.dhs.gov/schedulerapi is a public, unauthenticated endpoint with CORS headers that explicitly permit browser-based access. No terms of service prohibit automated access. The API is the same endpoint used by the official TTP Scheduler web application. NEXUS Alert functions identically to a user manually refreshing the appointment page — just faster and more reliably.
