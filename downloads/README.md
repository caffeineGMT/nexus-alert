# Lead Magnet PDFs

This directory contains downloadable lead magnet PDFs for email capture campaigns.

## Files Needed

1. **nexus-appointment-checklist.pdf**
   - Complete checklist for NEXUS appointment preparation
   - Documents required, payment methods, what to bring
   - Print-friendly format with checkboxes

2. **nexus-application-guide.pdf**
   - Step-by-step guide to applying for NEXUS
   - Timeline expectations, common pitfalls
   - Tips for successful application

3. **appointment-finding-tips.pdf**
   - 10 pro tips to find appointments faster
   - Best times to check for cancellations
   - How to use NEXUS Alert effectively

## Design Guidelines

- **Brand Colors:** #3b82f6 (primary blue), #22c55e (success green), #0a0a0a (dark background)
- **Typography:** Clean, professional sans-serif fonts
- **Size:** Keep PDFs under 2MB for fast downloads
- **Format:** Letter size (8.5" x 11") for easy printing

## Generation Tools

Use Canva, Figma, or Adobe InDesign to create professional-looking PDFs:
- Include NEXUS Alert branding/logo
- Add helpful infographics and checklists
- Make it visually appealing and easy to scan
