## 🆘 Troubleshooting

### Issue: Screenshots look pixelated on Chrome Web Store
**Solution:** Ensure you're capturing at 1280×800 with DPR = 2 (Retina quality). Export as PNG, not JPG.

### Issue: Video file size too large (>100MB)
**Solution:** Lower bitrate to 5-8 Mbps in export settings OR compress with Handbrake (free).

### Issue: Captions not syncing with voiceover
**Solution:** Check frame rate consistency (all clips 30fps). Import SRT file instead of manual text layers.

### Issue: Chrome Web Store video embed not loading
**Solution:** Ensure YouTube visibility is "Unlisted" (not Private). Check embed settings in YouTube.

### Issue: Extension features not working during screen capture
**Solution:** Ensure extension is loaded as unpacked (not packaged). Check background service worker is active in `chrome://extensions`.
