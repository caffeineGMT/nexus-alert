## 🛠️ Technical Recording Setup

### Screen Recording Settings
- **Software:** Loom (easiest), OBS Studio (advanced), or ScreenFlow (Mac)
- **Resolution:** 1920×1080 (1080p)
- **Frame Rate:** 30fps (60fps if showing smooth animations)
- **Audio:** 48kHz stereo, AAC codec
- **Bitrate:** 8-10 Mbps (high quality for YouTube)

### Browser Setup
- **Window size:** Full screen OR fixed 1280×800 (use DevTools device toolbar)
- **Zoom:** 100% (no scaling)
- **Extensions:** Hide all except NEXUS Alert
- **Bookmarks bar:** Hidden
- **Tabs:** Pre-loaded and ordered correctly

### Microphone Setup
- **Distance:** 6-8 inches from mouth
- **Pop filter:** Highly recommended (reduces plosives)
- **Recording level:** Peak at -6dB to -3dB (leave headroom)
- **Noise:** Record in quiet room, close windows, turn off AC/fan

## ✂️ Editing Workflow

### Step 1: Import Assets
1. Import all screen recordings
2. Import voiceover audio
3. Import background music
4. Import graphics (logo, tables, testimonials, end screen)
5. Organize in bins/folders

### Step 2: Rough Cut (Timeline Assembly)
1. Lay voiceover on Track 1 (anchor track)
2. Add screen recordings above voiceover (match timing)
3. Trim clips to exactly 60 seconds total
4. Add graphics layers (logo, tables, testimonials)
5. Verify smooth transitions between scenes

### Step 3: Captions (Hardcode with SRT)
1. Import `nexus-alert-demo-captions.srt`
2. Or manually add text layers using script above
3. Font: **Inter Bold** or **Montserrat Bold**
4. Size: **36-40px** (readable on mobile)
5. Color: **White** with **black outline** (3px) or drop shadow
6. Position: **Bottom third** of screen (safe zone)
7. Timing: Match voiceover exactly

### Step 4: Background Music
1. Import royalty-free music track (Epidemic Sound, Artlist, or Uppbeat)
2. Place on separate audio track
3. Fade in at 0:00 (0.5 sec fade)
4. Fade out at 0:58 (1 sec fade)
5. Mix at **40-50% volume** (voiceover should dominate)
6. Apply ducking: Lower music -6dB when voiceover plays

### Step 5: Sound Effects
1. Add notification chime at 0:32
2. Add button click sound at 0:38
3. Ensure effects are subtle (don't overpower voiceover)

### Step 6: Color Grading
1. Match colors across all clips (white balance, exposure)
2. Slight saturation boost (+10-15%) for vibrancy
3. Ensure brand colors accurate: Blue `#3b82f6`, Green `#22c55e`
4. Export color space: sRGB (web standard)

### Step 7: Final Export
**Export Settings:**
- Format: MP4 (H.264)
- Resolution: 1920×1080
- Frame rate: 30fps
- Bitrate: 8-10 Mbps
- Audio: AAC, 192kbps, stereo
- Duration: **Exactly 60 seconds** (±2 sec acceptable)
- File size: 50-100MB (YouTube-friendly)
