## Quality Assurance

### Testimonial Review Checklist

For each submission, verify:
- [ ] Real name provided (first + last)
- [ ] Location provided (city, state/province)
- [ ] Program specified (NEXUS/Global Entry/SENTRI)
- [ ] Specific results mentioned (time saved, slot found)
- [ ] Authentic language (not overly promotional)
- [ ] Permission granted for marketing use
- [ ] Email valid (for reward code delivery)

### Photo Quality Checklist

For photo submissions:
- [ ] Clear, high-resolution image
- [ ] Face visible (builds trust)
- [ ] Professional or casual (both acceptable)
- [ ] Permission granted for use
- [ ] Filename format: `[name]-[location].jpg`

### Webhook Setup

To auto-forward submissions to external tools:

1. **Google Sheets via Zapier:**
   - Create Zap: Webhooks by Zapier → Google Sheets
   - Get webhook URL: `https://hooks.zapier.com/hooks/catch/[ID]`
   - Set env var: `TESTIMONIAL_WEBHOOK_URL=[URL]`

2. **Airtable:**
   - Create base with testimonial fields
   - Use Airtable API or webhook integration
   - Auto-populate tracking sheet

3. **Email Notification:**
   - Forward submission data to hello@nexus-alert.com
   - Get real-time alerts for new testimonials
