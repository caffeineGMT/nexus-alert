## Crisis Management

### If Negative Coverage:
1. **Don't panic.** Negative press can be converted.
2. **Respond professionally** in comments/article
3. **Address legitimate concerns** with product updates
4. **Reach out privately** to journalist to clarify
5. **Use feedback** to improve product

### If Factual Errors:
1. **Email journalist immediately** with corrections
2. **Provide evidence** (screenshots, data)
3. **Ask for update/correction** politely
4. **Follow up** if not corrected within 48 hours

### If Journalist Asks Hard Questions:
1. **Be honest.** Don't dodge or obfuscate.
2. **Privacy concerns?** Explain local-only processing + open source code
3. **ToS concerns?** Explain we're using public API, not automating bookings
4. **Profitability?** Share revenue transparency (builds trust)

## Best Practices

### Email Pitching:
- ✅ **Personalize** every email (no mass BCC)
- ✅ **Keep it short** (<250 words for initial pitch)
- ✅ **Lead with value** (why their readers care)
- ✅ **Make it easy** (press kit link, clear CTA)
- ✅ **Follow up** (3 days, 7 days, 14 days)
- ❌ **Don't spam** (max 3 follow-ups)
- ❌ **Don't be pushy** (accept "no" gracefully)

### Interview Preparation:
- ✅ **Prepare talking points** (3-5 key messages)
- ✅ **Have stats ready** (user count, success rate, wait time reduction)
- ✅ **Be authentic** (personal story resonates)
- ✅ **Offer exclusive** (give journalist something special)
- ✅ **Follow up** (send thank-you + any promised materials)

### Relationship Building:
- ✅ **Think long-term** (this is first touch, not last)
- ✅ **Add value** (offer yourself as expert source)
- ✅ **Share their work** (promote their articles)
- ✅ **Keep them updated** (major milestones = re-pitch opportunity)

## FAQs for Media

### Common Questions Journalists Ask:

**Q: Is this against the GOES terms of service?**
A: No. NEXUS Alert queries the publicly available GOES appointment API — the same data you'd see manually checking the website. We're not automating bookings or circumventing security.

**Q: How do you make money if it's free?**
A: Freemium model. Free tier is fully functional. Premium tier ($4.99/mo) checks more frequently (every 2 min vs 30 min) and sends email/SMS alerts. About 15% of users upgrade.

**Q: What happens if GOES changes their API?**
A: We monitor for changes and update the extension as needed. All updates push through Chrome Web Store automatically.

**Q: Can users self-host this?**
A: Yes, it's open source on GitHub. Users can review code or run their own instance of the backend if desired.

**Q: What's your privacy policy?**
A: Free tier collects zero data — runs entirely in user's browser. Premium users share only email addresses for notifications. No tracking, no analytics in the extension.
