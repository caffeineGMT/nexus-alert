1. **Email verification:**
   - Use Hunter.io to find/verify email addresses
   - Format: firstname.lastname@domain.com OR first.last@domain.com
   - Fallback: info@domain.com, contact@domain.com

2. **LinkedIn research:**
   - Find decision-maker's LinkedIn profile
   - Check recent posts/activity (personalization fodder)
   - Note mutual connections (warm intro opportunity)

3. **Website audit:**
   - Check if they mention NEXUS/Global Entry specifically
   - Look for team bios (who handles NEXUS cases?)
   - Check blog/news for recent content (engagement opportunity)

4. **Tech stack analysis:**
   - Check if they use case management software (Clio, MyCase mention on site)
   - Look for tech-forward indicators (online scheduling, chatbot, etc.)
   - API integration opportunity if using Clio/MyCase

5. **Social media presence:**
   - Twitter/X activity (engagement opportunity)
   - Facebook business page (check reviews for NEXUS mentions)
   - YouTube channel (video content partnership potential)

## Notes & Strategy

**Multi-channel approach:**
- Send email + LinkedIn connection same day (increases touchpoints)
- If email bounces, try LinkedIn message as backup
- If LinkedIn ignored, phone call (if number available on website)

**Timing:**
- Send emails Tuesday-Thursday, 10am-11am or 2pm-3pm local time
- Send LinkedIn connections Monday-Friday, avoid weekends
- Schedule follow-ups 3-4 days apart (don't be too pushy)

**Personalization:**
- Reference their city/region ("I see you're near the Blaine NEXUS center...")
- Mention mutual connections if any
- Comment on a recent LinkedIn post before sending connection request

**Referral strategy:**
- If they say "not interested" → ask for referrals immediately
- If they start a trial → ask for 2-3 referrals after Week 1
- Offer referral incentive: "1 month free for every referral that converts"

**Partnership potential:**
- Identify influencers (Tahmina Watson, Evelyn Ackah, Jacob Sapochnick)
- Offer affiliate partnership: 20% recurring commission on referrals
- Feature them in case studies/testimonials if they convert
