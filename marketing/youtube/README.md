## Production Format

**Video Style:** Talking head + screen recording (50/50 split)
- Talking head: 1080p webcam, good lighting, clean background
- Screen recording: OBS Studio capturing ttp.cbp.dhs.gov + Chrome extension UI
- Editing: DaVinci Resolve or Adobe Premiere Pro
- Subtitles: Auto-generated via YouTube Studio + manual cleanup
- End screen: 20-second CTA overlay with extension install link + related video suggestions
