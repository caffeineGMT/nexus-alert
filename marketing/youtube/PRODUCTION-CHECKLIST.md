# YouTube Video Production Checklist

Use this checklist for each video to ensure consistent quality and SEO optimization.

---

## Pre-Production (Days 1-2)

### Script Preparation
- [ ] Read full script aloud (adjust awkward phrasing)
- [ ] Time script with teleprompter (target: 5-8 minutes)
- [ ] Print script with highlighted emphasis words
- [ ] Prepare B-roll shot list based on script mentions

### Equipment Check
- [ ] Camera: Clean lens, fully charged battery
- [ ] Microphone: Test audio levels (-12dB to -6dB)
- [ ] Lighting: 3-point setup (key, fill, back light)
- [ ] Teleprompter: Loaded with script, font size 36px+
- [ ] Screen recording software: OBS Studio configured (1920x1080, 60fps)

### Location Setup
- [ ] Background: Clean, non-distracting, good depth
- [ ] Room lighting: Natural light + supplemental lamps
- [ ] Eliminate background noise: Close windows, turn off AC/fans
- [ ] Test recording: 30-second camera test, check focus/audio

---

## Production Day (Days 3-4)

### Talking Head Recording
- [ ] Camera positioned at eye level (tripod)
- [ ] Frame composition: Rule of thirds (face left or right of center)
- [ ] White balance set correctly (manual mode, 5600K for daylight)
- [ ] Record 3 takes of full script (pick best sections in editing)
- [ ] Record B-roll inserts (hand gestures, reactions, cutaway shots)

### Screen Recording
- [ ] Close unnecessary tabs/apps (clean desktop)
- [ ] Browser zoom at 100% (no scaling issues)
- [ ] Mouse cursor: Yellow highlight or large pointer (visible in recording)
- [ ] Narrate while recording screen (later: sync with talking head)
- [ ] Record each demo section 2x (backup takes)

**Screen Recording Shots Needed:**
- [ ] Chrome Web Store search + install process
- [ ] Extension popup configuration
- [ ] CBP website navigation (ttp.cbp.dhs.gov)
- [ ] Simulated notification popup (can edit in post)
- [ ] Booking flow walkthrough
- [ ] Settings/dashboard views

---

## Post-Production (Days 5-7)

### Editing Software
- DaVinci Resolve (free) or Adobe Premiere Pro

### Editing Checklist
- [ ] Import all footage (talking head + screen recordings)
- [ ] Create timeline: 1920x1080, 30fps (or 60fps if screen recording)
- [ ] Sync talking head audio with screen recording footage
- [ ] Cut dead air, awkward pauses (keep pacing tight)
- [ ] Add B-roll to cover jump cuts
- [ ] Add intro (3-5 seconds max, logo animation)
- [ ] Add outro (20 seconds, end screen template)

### Visual Enhancements
- [ ] Add on-screen text for key points (lower thirds)
- [ ] Add arrows/highlights on screen recordings (important UI elements)
- [ ] Add zoom-in effects (when showing small details)
- [ ] Color grade talking head footage (consistency across shots)
- [ ] Add transitions (simple cuts or 0.5s cross-dissolve)

### Audio Optimization
- [ ] Remove background noise (Audacity noise reduction)
- [ ] Normalize audio levels (-3dB peak, no clipping)
- [ ] Add background music (royalty-free, 15% volume under voice)
- [ ] Fade music in/out during talking sections
- [ ] Add sound effects (notification chime, click sounds)

### Captions/Subtitles
- [ ] Auto-generate captions via YouTube Studio (after upload)
- [ ] Download .SRT file and manually correct errors
- [ ] Re-upload corrected captions
- [ ] Add styled captions for key quotes (in-video text, not just subtitles)

### End Screen Elements (Last 20 seconds)
- [ ] "Subscribe" button (animated)
- [ ] "Install Extension" CTA with link
- [ ] 2 related video suggestions (clickable thumbnails)
- [ ] Background: Solid color or looping B-roll

---

## Thumbnail Creation (Day 7)

### Canva Design
- [ ] Use template from `thumbnail-templates.md`
- [ ] Export as PNG, 1280x720px
- [ ] File size under 2MB
- [ ] Test thumbnail at small size (readable on mobile?)

### A/B Testing Setup
- [ ] Create 2 thumbnail variants
- [ ] Save both as `thumbnail-A.png` and `thumbnail-B.png`
- [ ] Plan to swap thumbnails after 24 hours (test CTR)

---

## YouTube Upload (Day 8)

### Video File
- [ ] Final export: MP4, H.264 codec, 1920x1080, 30fps
- [ ] File size: Under 128GB (YouTube limit)
- [ ] Test playback locally (no audio sync issues)

### Upload Settings
- [ ] Title: Copy from `titles-descriptions-tags.md` (optimize for SEO)
- [ ] Description: Full description with timestamps, links, hashtags
- [ ] Tags: All tags from SEO doc (under 500 characters)
- [ ] Thumbnail: Upload `thumbnail-A.png`
- [ ] Playlist: Add to "NEXUS Alert Tutorials" playlist
- [ ] Category: "Howto & Style" or "Education"
- [ ] Language: English
- [ ] License: Standard YouTube License

### Visibility Settings
- [ ] **Option A (Scheduled Release):** Schedule for Tuesday or Thursday, 10 AM PT
- [ ] **Option B (Immediate Release):** Set to Public immediately

### Monetization (If Eligible)
- [ ] Enable ads (mid-roll ads every 5 minutes for 8+ min videos)
- [ ] Content declaration: No paid promotion, no product placement

### Advanced Settings
- [ ] Comments: Enabled, hold potentially inappropriate for review
- [ ] Ratings: Show like/dislike counts
- [ ] Video location: Not set (irrelevant for this content)
- [ ] Recording date: Not set
- [ ] License: Standard YouTube License
- [ ] Distribution: Allow embedding, publish to subscriptions feed

## Engagement (Days 8-30)

### Daily Tasks (First 7 Days)
- [ ] Reply to ALL comments within 24 hours (boosts engagement signal)
- [ ] Heart relevant comments (shows appreciation)
- [ ] Pin best comment (if better than original pinned comment)

### Weekly Tasks
- [ ] Check YouTube Studio analytics (views, CTR, retention)
- [ ] Update performance tracking spreadsheet
- [ ] A/B test thumbnail if CTR < 3% (swap to thumbnail-B.png)
- [ ] Promote in communities where video is gaining traction

### Monthly Tasks
- [ ] Create community post: "Thanks for 10K views! What topic should I cover next?"
- [ ] Use top comments as inspiration for next video
- [ ] Review retention graph → identify drop-off points → improve for next video

---

## Success Metrics (30-Day Review)

Target Goals for Each Video:
- [ ] **Views:** 1,000+ (first week), 10,000+ (90 days)
- [ ] **CTR:** 5%+ (if lower, revise thumbnail/title)
- [ ] **Avg View Duration:** 50%+ (3+ minutes for 6-min video)
- [ ] **Likes:** 4%+ of views (400 likes per 10K views)
- [ ] **Comments:** 1%+ of views (100 comments per 10K views)
- [ ] **Extension Installs:** 500+ (5% of 10K viewers)

If Not Meeting Goals:
- [ ] Review analytics → identify issue (low CTR? low retention?)
- [ ] Implement corrective action (new thumbnail, tighter script, better hook)
- [ ] Apply learnings to next video in series

## Equipment & Software Recommendations

### Free Tools
- **Screen Recording:** OBS Studio (free, open-source)
- **Video Editing:** DaVinci Resolve (free version)
- **Audio Editing:** Audacity (noise removal, normalization)
- **Thumbnails:** Canva Free (1280x720 templates)
- **Captions:** YouTube Studio (auto-generate + manual correction)

### Paid Tools (Optional)
- **Video Editing:** Adobe Premiere Pro ($22.99/month)
- **Thumbnails:** Canva Pro ($12.99/month - background remover, brand kit)
- **Music:** Epidemic Sound ($15/month - royalty-free library)
- **SEO Research:** TubeBuddy ($9/month - keyword research, A/B testing)

### Hardware (Recommended Minimum)
- **Camera:** iPhone 12+ or webcam (Logitech C920 or better)
- **Microphone:** Blue Yeti USB mic ($100) or lavalier mic ($30)
- **Lighting:** 2x softbox lights ($50 on Amazon) or natural window light
- **Teleprompter:** iPad with PromptSmart app (free) or laptop + CuePrompter web app

## Final Pre-Publish Checklist

Before clicking "Publish":
- [ ] Title is SEO-optimized and under 70 characters
- [ ] Description has link in first 2 lines + timestamps + hashtags
- [ ] All tags added (check character limit)
- [ ] Thumbnail uploaded and looks good in preview
- [ ] End screen elements configured (subscribe, related videos, extension link)
- [ ] Captions uploaded and error-free
- [ ] Video added to "NEXUS Alert Tutorials" playlist
- [ ] Scheduled publish time set (Tuesday/Thursday 10 AM PT) OR publishing immediately
- [ ] Pinned comment drafted and ready to post immediately after publish

**Hit Publish. Then:**
- [ ] Post pinned comment
- [ ] Share on Reddit (5 subreddits)
- [ ] Share on FlyerTalk
- [ ] Share on social media
- [ ] Add to email newsletter
- [ ] Monitor comments for first 2 hours (reply immediately for engagement boost)
