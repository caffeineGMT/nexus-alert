# YouTube Thumbnail Design Specifications

All thumbnails optimized for 1280x720px (16:9 ratio) with high-contrast text for mobile visibility.

---

## Design Principles

### Color Palette
- **Primary:** Bold red (#FF0000) or bright blue (#0066FF) for urgency
- **Secondary:** White text with 4px black stroke for readability
- **Background:** High-contrast gradient or blurred screenshot
- **Accent:** Green (#00FF00) for success metrics, yellow (#FFD700) for attention

### Typography
- **Headline font:** Montserrat Black or Impact (bold, sans-serif)
- **Body font:** Open Sans Bold
- **Minimum size:** 60px for main text (readable at 320px width)
- **Stroke:** 4-6px black outline on all text
- **Drop shadow:** 50% opacity, 4px offset

### Composition
- **Rule of thirds:** Place focal point on intersection
- **Face placement:** If using talking head, position face left or right, not center
- **Text hierarchy:** 70% headline, 30% subtext
- **Whitespace:** 15% minimum border padding

## Thumbnail Testing Checklist

Before finalizing, test each thumbnail:

- [ ] **Mobile view:** Shrink to 320px width - is text still readable?
- [ ] **Color contrast:** Does text pop against background (4.5:1 minimum ratio)?
- [ ] **Face recognition:** If using faces, are expressions clear at small size?
- [ ] **Focal point:** Does eye track to main element within 0.5 seconds?
- [ ] **Consistency:** Does it match brand style across video series?
- [ ] **A/B testing:** Create 2 variants per video, test for 48 hours
- [ ] **File format:** Export as PNG (no compression artifacts)
- [ ] **Dimensions:** Exactly 1280x720px (no stretching or letterboxing)
- [ ] **File size:** Under 2MB (YouTube limit)

## Color Psychology

- **Red:** Urgency, wasted time, frustration (use for "manual" comparisons)
- **Green:** Success, speed, efficiency (use for "automated" results)
- **Blue:** Trust, technology, professional (use for tutorial/guide videos)
- **Yellow:** Attention, highlight, price (use for CTAs and pricing)
- **White + Black Stroke:** Maximum readability across all backgrounds
