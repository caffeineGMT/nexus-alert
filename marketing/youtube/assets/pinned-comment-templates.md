## Pinned Comment Best Practices

### Structure
1. **Greeting:** 👋 emoji + friendly opening
2. **Primary CTA:** Install link (UTM tagged)
3. **Value prop:** 3 bullet points max
4. **Discount code:** Creates urgency
5. **Question:** Encourages engagement

### Emojis to Use
- 👋 Greeting
- 🔔 Notification/install action
- ✅ Checklist/benefits
- 💰 Pricing
- 💎 Premium tier
- ⚡ Speed/urgency
- 💡 Pro tip
- 🎁 Discount/offer
- 📊 Data/results
- ⏱️ Time-related

### Timing
- Post pinned comment **immediately** after video goes live
- YouTube algorithm prioritizes early engagement
- First 2 hours are critical for comment visibility

### Engagement Tactics
- Ask a question (encourages replies)
- Create a poll using emoji reactions
- Offer exclusive discount code (YOUTUBE10)
- Request video topic suggestions
- Respond to replies within 1 hour (boosts engagement)

### Link Formatting
Always use full URL with UTM parameters:
```
https://chrome.google.com/webstore/detail/nexus-alert/[EXTENSION-ID]?utm_source=youtube&utm_medium=video&utm_campaign=[VIDEO-SLUG]
```

### A/B Testing Pinned Comments
After 7 days, if install CTR < 3%:
1. Unpin original comment
2. Post new pinned comment with different structure (e.g., social proof focused)
3. Compare install rates over next 7 days
4. Keep winner

### Analytics Tracking
Monitor in Google Analytics:
- **UTM source:** youtube
- **UTM medium:** video
- **UTM campaign:** [video-slug]

Track conversion funnel:
1. YouTube view
2. Pinned comment link click
3. Chrome Web Store page view
4. Extension install

Target: 5% of YouTube viewers click pinned comment link.
