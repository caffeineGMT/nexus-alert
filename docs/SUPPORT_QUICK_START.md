## ✅ Week 1: Supervised Support (Ongoing)

### Daily Routine

**Morning (30 min):**
- [ ] Check email inbox (respond to overnight tickets)
- [ ] Review Crisp queue
- [ ] Prioritize: Premium users → Critical bugs → Free users
- [ ] Set status to "Available" in Crisp

**Throughout Day:**
- [ ] Respond to new tickets within SLA (Free: 48h, Premium: 24h)
- [ ] Use canned responses but personalize
- [ ] Link to help articles in every response
- [ ] Escalate anything you can't resolve in 15 min (it's okay!)
- [ ] Document new issues you encounter

**End of Day (15 min):**
- [ ] Clear email inbox (respond or defer with acknowledgment)
- [ ] Set Crisp status to "Away"
- [ ] Write handoff notes if multi-person team
- [ ] Log any bugs found in escalation tracker

## 📚 Quick Reference Cheat Sheet

### Common Issues & Fixes (80% of tickets)

**1. Not Getting Notifications:**
```
→ Check permissions: chrome://settings/content/notifications
→ Verify monitoring is ON (extension popup)
→ Check Settings → Desktop Notifications enabled
→ Free tier: 30-min checks (explain upgrade)
```

**2. Extension Not Installing:**
```
→ Chrome version ≥ 88? (Settings → About Chrome)
→ From Chrome Web Store, not sideload
→ Corporate device? May be blocked by IT
→ Try: Clear cache, restart Chrome, retry
```

**3. Upgrade to Premium:**
```
→ Click extension icon → "Upgrade to Premium"
→ OR visit: https://nexus-alert.com/pro
→ Benefits: 2-min checks, email/SMS, priority support
→ Pricing: $4.99/mo or $49.99/yr
```

**4. Refund Request:**
```
→ Within 30 days? → APPROVE
→ Gather: Email, purchase date
→ Stripe → Refund → 5-10 business days
→ Gather feedback (optional but valuable)
```

**5. Cancellation:**
```
→ Stripe Customer Portal link (in billing emails)
→ OR walk them through: Dashboard → Subscriptions → Cancel
→ Explain: Keep Premium until billing period ends
→ Then auto-downgrade to Free tier
```

## 🎯 Your First Response Checklist

Before hitting "Send" on ANY support response:

- [ ] **Did I read their entire message?** (Don't skim!)
- [ ] **Is this Free or Premium user?** (Prioritize Premium)
- [ ] **Did I personalize it?** (Use their name, reference their issue)
- [ ] **Are instructions clear?** (Step-by-step, no jargon)
- [ ] **Did I link to a help article?** (Always!)
- [ ] **Did I set expectations?** (Timeline if escalating)
- [ ] **Is my tone empathetic?** (Acknowledge frustration)
- [ ] **Did I end with a call-to-action?** ("Let me know if this helps!")
- [ ] **Proofread?** (No typos!)
