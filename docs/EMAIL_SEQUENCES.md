## Implementation Notes

1. **Timing:** Use ConvertKit's automation builder to set delays between emails
2. **Segmentation:** Tag users based on tier (free/premium) to send targeted content
3. **A/B Testing:** Test subject lines with emoji vs plain text
4. **Personalization:** Use merge tags for {first_name}, {tier}, {location}
5. **Compliance:** Include unsubscribe link in all marketing emails (not transactional)

For setup instructions, see `docs/EMAIL_MARKETING_SETUP.md`.
