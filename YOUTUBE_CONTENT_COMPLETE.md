### ✅ Production Checklist

**Location:** `marketing/youtube/PRODUCTION-CHECKLIST.md`

**Complete step-by-step checklist for each video:**

**Pre-Production (Days 1-2):**
- Script preparation (read aloud, time with teleprompter)
- Equipment check (camera, microphone, lighting, OBS Studio)
- Location setup (background, lighting, eliminate noise)

**Production Day (Days 3-4):**
- Talking head recording (3 takes, rule of thirds framing)
- Screen recording (1920x1080, 60fps, clean desktop)
- B-roll shots (hand gestures, reactions, cutaways)

**Post-Production (Days 5-7):**
- Editing (DaVinci Resolve or Premiere Pro)
- Visual enhancements (on-screen text, arrows, zoom effects)
- Audio optimization (noise reduction, normalization, background music)
- Captions/subtitles (auto-generate → manual correction)
- End screen elements (subscribe button + 2 related videos + install CTA)

**Upload Day (Day 8):**
- Title, description, tags from SEO doc
- Thumbnail upload (A/B test variants)
- End screen configuration
- Publish or schedule (Tuesday/Thursday 10 AM PT)
- Post pinned comment immediately
- Distribution (Reddit, FlyerTalk, social media)

**Post-Upload Optimization:**
- Reply to all comments within 24 hours
- A/B test thumbnails if CTR < 3%
- Weekly analytics review
- Monthly optimization based on retention data
